# Website for Automated Testing with Cypress

> This is an open-source demo website independently designed, implemented, and maintained by the author for practicing and demonstrating automated testing with Cypress.

> The project is inspired by the Conduit blogging platform (a Medium.com clone) but is a fully original, standalone implementation created specifically as a test automation playground. It is not affiliated with the original Conduit project.

---

> Это open-source демо-сайт, самостоятельно спроектированный, реализованный и поддерживаемый автором для практики и демонстрации автоматизированного тестирования с использованием Cypress.

> Проект вдохновлён блог-платформой Conduit (клон Medium.com), но представляет собой полностью оригинальную и независимую реализацию, созданную специально как playground для автотестов. Проект не связан с оригинальным Conduit.

### [See Demo](https://cypress-testing-website.vercel.app/)

* My Cypress automation test code is available in a separate repository [here](https://github.com/psiyavush/cypress-autotests-for-conduit)

---

* Мой проект кода автотестов на Cypress находится в отдельном репозитории [здесь](https://github.com/psiyavush/cypress-autotests-for-conduit)

## For local development

```bash
npm install

```

